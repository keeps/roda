Event logs are special files that record significant events on the repository, such as when a user logs on to, when a user downloads a file, when a user updates a descriptive metadata file. Whenever these types of events occur, the repository records the event in the event log that you can read by using this page.

For each event the following information is recorded: date, involved compomnent, method, involved objects, user that executed the action, the duration, and the IP address of the user that executed the action.

Users are able to filter events by type, date and other attributes by selecting the options available on the right side panel.
