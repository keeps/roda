Auf dieser Seite können Sie die Entsorgungsbestätigungen einsehen, die für dieses Repository erstellt wurden. Eine Entsorgungsbestätigung besteht aus einem Bericht, der die intellektuellen Entitäten und zusätzliche Metadateninformationen zusammenfasst. Um die mit einer Entsorgungsbestätigung verknüpften intellektuellen Entitäten zu vernichten, müssen Sie explizit die Aktion "Vernichten" ausführen. Nach der Zerstörung können die intellektuellen Entitäten innerhalb der Entsorgungsbestätigung wiederhergestellt oder dauerhaft gelöscht werden.
