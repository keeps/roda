Ajuda comunitária

Há duas mailing-lists que o poderão ajudar a compreender melhor o RODA. Uma é para utilizadores do sistema e a outra dedicada a programadores.

Mailing-list de utilizadores

A lista de email dos utilizadores é para todos os que utilizam o RODA, como arquivistas, produtores ou gestores de arquivo que querem partilhar problemas na utilização das funcionalidade do RODA. Inscreva-se nesta lista de email para fazer perguntas ou participar em discussões sobre todo os aspectos da instalação, execução e utilização do RODA.

Lista de email para programadores

A lista de email para programadores serve para todos que querem desenvolver novas funcionalidades no RODA. É o lugar para partilhar ideias e discutir modificações no código open-source. Faça parte da equipa de desenvolvimento e ajude a moldar a evolução do software RODA. A comunidade RODA depende dos seu membros para definir o desenho, teste e documentação do projeto.
