El servicio de gestión de usuarios permite al administrador de repositorio crear o modificar las credenciales de acceso para cada usuario en el sistema. Este servicio también permite al administrador definir grupos y permisos para cada uno de los usuarios registrados. Los administradores también pueden filtrar los usuarios y grupos actuales que se muestra haciendo clic en las opciones disponibles en el panel lateral derecho. Para crear un nuevo usuario, haga clic en el botón "Añadir usuario" . Para crear un nuevo grupo de usuarios, haga clic en el botón "Agregar grupo". Para editar un usuario o grupo exietnte, haga clic en un elemento de la tabla de abajo.
