O relatório de riscos lista todos os riscos identificados que podem afetar o repositório. Aqui pode gerir os riscos de modo à lista ser o mais completa possível e incluir todas as ameaças que tenham uma probabilidade de acontecer mensurável. A mitigação de riscos é o processo de definir acções de preservação para reduzir a probabilidade ou o impacto das ameaças nos objetivos do repositório.
