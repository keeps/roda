O serviço de gestão de utilizadores permite ao gestor do repositório criar ou modificar as credenciais de acesso de cada utilizador no sistema. Este serviço também permite ao gestor definir grupos e permissões para cada um dos utilizadores registados. Os gestores podem ainda filtrar os utilizadores e grupos atualmente visíveis, clicando nas opções disponíveis no painel lateral direito. Para criar um novo utilizador, clique no botão "Novo utilizador". Para criar um novo grupo de utilizadores, clique no botão "Novo grupo".
