En bevarandehändelse samlar metadata om åtgärder, specifikt dokumenterar vilka objekt den påverkar och vilka mänskliga eller mjukvaru-agenter som ingrep. Dokumentation av åtgärder som modifierar ett objekt är avgörande för att upprätthålla digital härkomst, en nyckelfaktor för äkthet. Åtgärder som skapar nya relationer eller ändrar befintliga relationer är viktiga för att förklara dessa relationer. Även handlingar som inte förändrar någonting, såsom giltighet och integritetskontroller av objekt, kan vara viktiga att registrera för förvaltningsändamål.
