Nesta página é possivel alterar a ordem das condições de eliminação. É nesta página que é definida a prioridade com que as condições de eliminação são aplicadas ao repositório. A primeira posição na tabela corresponde ao nível máximo de prioridade, enquanto que a última posição na tabela corresponde ao nível mínimo de prioridade. Quando uma entidade intelectual nao é abrangida por nenhuma condição de eliminação, nenhuma tabela de seleção é associada.
