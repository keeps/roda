Representationsinformation är all information som krävs för att kunna förstå och tolka både det digitala materialet samt tillhörande metadata. Digitala objekt lagras som bitströmmar som inte kan förstås av en människa om det inte finns ytterligare data för att tolka dem. Representationsinformation är den extra strukturella eller semantiska informationen som omvandlar rå data till något mer meningsfullt. Den här sidan ger bevarandespecialisten möjlighet att definiera representationsinformation och att länka den till intellektuella entiteter i e-arkivet.
