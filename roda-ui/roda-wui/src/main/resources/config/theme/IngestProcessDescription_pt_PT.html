O processo de Ingestão inclui serviços e funções para aceitar Pacotes de Informação de Submissão (SIPs) dos Produtores, preparar Pacotes de Informação Arquivística (AIPs) para armazenamento e garantir que os AIPs e a respetiva Informação Descritiva de apoio fiquem devidamente estabelecidos no repositório. Esta página apresenta todos os trabalhos de ingestão que estão atualmente a ser executados, bem como todos os que foram realizados no passado. No painel lateral direito, é possível filtrar os trabalhos com base no seu estado, no utilizador que iniciou o trabalho e na data de início. Ao clicar num item da tabela, é possível ver o progresso do trabalho, assim como detalhes adicionais.
