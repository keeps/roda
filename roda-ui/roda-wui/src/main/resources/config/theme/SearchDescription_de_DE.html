Auf dieser Seite können Sie nach geistigen Entitäten, Repräsentationen oder Dateien suchen (verwenden Sie den Pfeil nach unten, um das Suchdomäne auszuwählen). Für jede dieser Domänen können Sie in allen Eigenschaften oder in bestimmten Eigenschaften suchen (verwenden Sie den Pfeil nach unten, um die erweiterte Suche zu öffnen). Wenn Sie beispielsweise Intellektuelle Entitäten auswählen, können Sie in einem bestimmten Feld der beschreibenden Metadaten suchen oder Dateien eines bestimmten Formats finden, wenn die erweiterte Suche für Dateien ausgewählt ist.

Die Suchmaschine findet nur ganze Wörter. Wenn Sie nach teilweisen Begriffen suchen möchten, sollten Sie den '*' Operator verwenden. Weitere Informationen zu den verfügbaren Suchoperatoren finden Sie in der help page.
