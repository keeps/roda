Beziehung zu einer intellektuellen Entität

Definieren Sie eine Beziehung zwischen dieser Repräsentationsinformation und einer geistigen Entität. Der Grund für die Beziehung kann in der kontrollierten Vokabulare des Beziehungstyps beschrieben werden. Sie können die spezifische intellektuelle Entität auswählen, die von diesem Repository verwaltet wird, und einen Titel für die Beziehung definieren, der als Text des erstellten Links angezeigt wird.
