Beziehung zu einer Webseite

Definieren Sie eine Beziehung zwischen dieser Repräsentationsinformation und einer Webseite. Der Grund für die Beziehung kann in der kontrollierten Vokabulare des Beziehungstyps beschrieben werden. Sie können den Link zur Webseite auswählen (z. B. http://roda-community.org) und einen Titel für die Beziehung definieren, der als Text des erstellten Links angezeigt wird.
