På den här sidan kan du se de logiska enheter där gallringsfristen har förfallit eller de logiska enheter som är redo för granskning. Om den logiska enheten är redo för gallring kan gallringen genomföras. Alternativt kan man göra en justering i gallringsschemat. Om den logiska enhetern är redo för granskning kan man endast ändra i gallringsschemat. Det är också möjligt att se de logiska enheter där uträkningen av gallringsfristen har misslyckats.
