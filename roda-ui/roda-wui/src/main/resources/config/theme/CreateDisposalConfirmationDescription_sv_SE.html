På den här sidan kan du se de objekt där gallringsfristen har förfallit eller där arkivobjekten är redo för granskning. Om arkivobjekten är redo för gallring kan gallringen genomföras. Alternativt kan man göra en justering i gallringsschemat. Om arkivobjekten är klara för granskning kan man endast ändra i gallringsschemat. Det är också möjligt att se de arkivobjekt där uträkningen av lagringstiden/gallringsfristen har misslyckats.
