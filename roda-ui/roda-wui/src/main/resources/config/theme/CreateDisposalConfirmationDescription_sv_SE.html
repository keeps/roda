På den här sidan kan du se de enheter där gallringsfristen är förfallen, antingen för att gallra eller för att granska inför gallring. Om arkivobjekten är redo för gallring går det antingen bra att genomföra gallringen eller justera gallringsschemat. Om arkivobjekten är klara för granskning går det bra att justera gallringsschemat. Det går också att få en överblick på gallringsschema som behöver justeras.
