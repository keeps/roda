Inleveransprocessen innehåller tjänster och funktioner för att godkänna Submission Information Packages (SIP:ar) från Producenter, förbereda arkivinformationspaket (AIP:er) för lagring och för att säkerställa att AIP:erna och dess beskrivande metadata blir säkra i systemet. Den här sidan listar alla inleveransjobb som pågår och alla som har genomförts. På panelen till höger finns det möjlighet att filtrera listan baserat på status, vem som startat körningen och startdatum. Genom att klicka på en post i tabellen kan man se körningsprocessen samt få ytterligare information.
