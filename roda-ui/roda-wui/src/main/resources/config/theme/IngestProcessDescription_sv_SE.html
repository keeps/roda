Processen för Ingest (inleverans) innehåller tjänster och funktioner för att acceptera Submission Information Packages (SIP:ar) från Producenter, förbereda arkiv Information Packages (AIP:er) för lagring och för att säkerställa att AIP:erna och dess beskrivande metadata blir tillgängliga i systemet. Den här sidan listar alla inleveranser som pågår och alla som har genomförts. På panelen till höger finns det möjlighet att filtrera listan baserat på status, vem som startat körningen och datum. Genom att klicka på respektive körning, går det att få ytterligare detaljer.
