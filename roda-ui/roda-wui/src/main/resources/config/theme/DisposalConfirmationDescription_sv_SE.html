På den här sidan kan du titta på de gallringsbekräftelser som har skapats för systemet. En gallringsbekräftelse innehåller en rapport med metadata och tillhörande objekt. För att gallra den intellektuella enheten som är associerad med gallringsbekräftelsen, måste du exekvera gallringen. Efter gallring kan den intellektuella enheten inom gallringsbekräftelsen antingen återställas eller permanent förstöras.
