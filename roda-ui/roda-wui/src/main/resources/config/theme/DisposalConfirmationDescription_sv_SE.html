På den här sidan kan du se de gallringsbekräftelser som har skapats för detta arkiv. En gallringsbekräftelse innehåller en rapport som samlar de logiska enheterna och extra metadatainformation. För att gallra den logiska enhet som är associerat med gallringsbekräftelsen, måste du explicit utföra gallringsåtgärden. Efter gallring kan den logiska enheten antingen återställas eller tas bort permanent.
