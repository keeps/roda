Repositorio digital de código abierto diseñado para la preservación

RODA (Repositorio de Registros Digitales Auténticos) es una solución de repositorio digital a largo plazo que proporciona funcionalidades para todas las unidades funcionales principales del modelo de referencia OAIS.

RODA es capaz de ingestar, gestionar y proporcionar acceso a varios tipos de contenido digital producido por grandes corporaciones y organismos públicos. RODA fue desarrollado utilizando tecnologías de código abierto y está respaldado por los siguientes estándares:

Autenticidad

RODA utiliza metadatos de preservación PREMIS para crear una cadena de confianza entre todas las generaciones de activos digitales.

Los metadatos de preservación, junto con el establecimiento de confianza en el entorno circundante (ISO 16363 e ISO 14721) aseguran que el servicio de repositorio sea fiable y los activos digitales permanezcan auténticos a lo largo del tiempo.

RODA también admite plugins que evalúan la validez de las firmas digitales y vuelven a firmar archivos PDF archivados cuando la validez de las firmas digitales está próxima a expirar.

Independencia del proveedor

RODA está construido sobre tecnologías de código abierto robustas y duraderas.

Toda la infraestructura necesaria para soportar RODA también es independiente del proveedor. Esto significa que puede utilizar el hardware y la distribución de Linux que mejor se adapten a las necesidades de su institución.

Como el producto es de código abierto, no es necesario depender de un solo proveedor para el soporte. Todo el código fuente del sistema está disponible en GitHub para su inspección.

Conformidad con estándares

RODA cumple con varios estándares abiertos de metadatos descriptivos como EAD y Dublin Core, PREMIS para metadatos de preservación y METS para metadatos estructurales.

Acciones de preservación

Las acciones de preservación pueden ejecutarse directamente desde la interfaz de usuario sobre cualquier selección de objetos digitales en el repositorio.

Retención y eliminación

RODA incorpora funcionalidad robusta y completa para la definición e implementación de políticas de retención y eliminación.

Escalabilidad

RODA es ideal para grandes organizaciones con millones de objetos digitales. Su naturaleza orientada a servicios permite una alta escalabilidad, posibilitando la distribución de la carga entre varios servidores.

Flujos de ingesta

Los flujos de ingesta pueden ser personalizados por el usuario para implementar políticas institucionales y gestionar colecciones especiales de datos. Los flujos de ingesta pueden mejorarse mediante el uso de plugins de código abierto o comerciales.

Adaptabilidad

La arquitectura modular de RODA facilita la adición de más funcionalidades al sistema sin afectar su funcionamiento principal.

Ediciones Enterprise

RODA es una solución de código abierto, lo que significa que cualquier persona puede descargar el código fuente, compilarlo y tenerlo funcionando en su propia institución en cuestión de horas.

Aunque esta flexibilidad es una gran ventaja, las organizaciones también pueden beneficiarse de la experiencia de especialistas en TI, software y preservación digital para adaptar la solución a sus necesidades únicas añadiendo funcionalidades e integraciones especializadas.

KEEP SOLUTIONS, el mantenedor del proyecto de código abierto RODA, ofrece tres ediciones comerciales especialmente diseñadas para satisfacer las exigencias de instituciones medianas y grandes con amplias colecciones de registros digitales, proporcionando la capacidad de gestionar activos digitales de manera eficiente en entornos de producción a gran escala.

Estas ediciones se llaman "Enterprise", "Enterprise HA" y "Enterprise SM".
