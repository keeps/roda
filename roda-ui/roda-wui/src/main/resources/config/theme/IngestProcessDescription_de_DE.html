Der Ingest Prozess umfasst Dienste und Funktionen, um SIPs von Produzenten zu akzeptieren AIPs für die Speicherung vorzubereiten und AIPs und die unterstützenden beschreibenden Informationen in das Repository aufzunehmen. Diese Seite listet alle Ingest-Jobs, die derzeit ausgeführt werden, sowie die in der Vergangenheit ausgeführt wurden. Auf der rechten Seite ist es möglich, Aufträge nach Status, Benutzer und Startdatum zu filtern. Wenn Sie auf ein Element in der Tabelle klicken, können Sie den Fortschritt des Auftrags sowie weitere Details sehen.
