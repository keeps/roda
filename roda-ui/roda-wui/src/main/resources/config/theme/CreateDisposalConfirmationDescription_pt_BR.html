Nesta página pode ver as entidades cujo prazo de retenção expirou. Se as entidades intelectuais estão prontas para eliminação é possivel associá-las a um novo auto de eliminação ou alterar a sua tabela de seleção. Se as entidades intelectuais estão prontas para revisão, apenas é possivel alterar a sua tabela de seleção. Além disso, tambem é possivel ver entidades intelectuais cujo cálculo do prazo de convervação tenha falhado.
