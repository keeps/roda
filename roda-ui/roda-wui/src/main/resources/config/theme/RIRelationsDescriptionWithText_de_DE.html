Beziehung zu einem anderen Ressourcentyp

Definieren Sie eine Beziehung zwischen dieser Darstellungsinformation und einer anderen Art von Ressource (z. B. einem Buch). Der Grund für die Beziehung kann im Beziehungstyp "Kontrolliertes Vokabular" beschrieben werden, Sie können die andere Ressource mit Text identifizieren (z. B. mit einer bibliografischen Referenz).
