Kapcsolat más típusú forrásokkal

Határozza meg a kapcsolatot a eprezentációs információk között és más típusú erőforrás (pl. egy könyv) között. Az ok a kapcsolat oka az ellenőrzött kapcsolattípusban írható le szótárban, a másik erőforrást szöveggel azonosíthatja (pl. egy bibliográfiai hivatkozással).
