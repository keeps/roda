Gallringsregler är ett antal regler och krav som sätter upp gallringscheman för varje intellektuell enhet i systemet. Gallringsreglerna kan sättas upp när som helst för att se till att systemet hålls konsekvent. Gallringsregler kan också sättas upp under inleveransprocessen. Gallringsreglerna har egenskaper och regler som triggar att gallringen genomförs eller inte. Om ett arkivobjekt inte uppfyller någon av dessa regler kommer den inte att följa gallringsregeln.
