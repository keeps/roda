Gallringsregler är ett antal regler och krav som sätter gallringsschema för varje intellektuell enhet i systemet. Gallringsreglerna kan sättas när som helst för att säkerställa upprätthållandet av systemet. Gallringsregler kan också sättas under inleveransprocessen. Gallringsreglerna har egenskaper och regler som triggar att gallringen genomförs eller inte. Om ett arkivobjekt inte uppfyller någon av dessa regler kommer den inte att följa gallringsregeln.
