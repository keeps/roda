Szellemi entitással való kapcsolat

Határozza meg a kapcsolatot a(z) reprezentációs információ és az intellektuális entitás között. A kapcsolat oka leírható a kapcsolat típusa által ellenőrzött szókészletben, kiválaszthatja az adott repozitórium által kezelt speciális szellemi entitást, és definiálhat címet a kapcsolathoz, amely a létrehozott hivatkozás szövegeként jelenik meg.
