Ankomstkontroll är processen för att avgöra om arkivobjekten och annat material har bevarandevärde. Bedömning kan göras på samlings-, skapar-, serie-, fil- eller objektsnivå. Bedömningen kan ske före överföringen och före den fysiska flytten, under eller efter anslutningen. Grunden för bedömningsbeslut kan omfatta ett antal faktorer inklusive informationens härkomst, innehåll, autenticitet, tillförlitlighet, ordning, fullständighet, skick, kostnader för bevarandet samt informationens egenvärde.
