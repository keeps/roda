As ações de preservação são tarefas realizadas sobre os conteúdos do repositório com o objetivo de melhorar a acessibilidade dos ficheiros arquivados ou mitigar riscos relacionados com a preservação digital. No RODA, as ações de preservação são geridas por um módulo de execução de tarefas. Este módulo permite ao gestor do repositório executar ações sobre um determinado conjunto de dados (AIPs, representações ou ficheiros). As ações de preservação incluem conversões de formato, verificações de checksums, geração de relatórios (por exemplo, para envio automático de emails de aceitação/rejeição de SIPs), verificações de vírus, entre outras.
