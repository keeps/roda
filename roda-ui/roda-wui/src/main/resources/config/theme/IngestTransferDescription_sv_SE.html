Överföring tillhandahåller tillfällig lagring för ta emot inlämningsinformationspaket (SIP) från producenter. SIP:ar kan levereras via elektronisk överföring (t.ex. FTP) eller laddas från media och bifogas till systemet. Denna sida gör det också möjligt för användaren att söka bland filer som finns i den tillfälliga överföringen, skapa/ta bort mappar och ladda upp flera SIP:ar till systemet samtidigt för vidare bearbetning och inleverans. Inleveransprocessen kan initieras genom att välja SIP:arna du vill inkludera i processen. Klicka på processknappen för att initiera intagsprocessen.
