Inleveransytan är en tillfällig lagringsyta för att ta emot inlämningsinformationspaket (SIP) från producenter. SIP:ar kan levereras via elektronisk överföring (t.ex. FTP) eller laddas från media kopplade till systemet. Denna sida gör det också möjligt för användaren att söka bland filer som finns i den tillfälliga överföringsarean, skapa/ta bort mappar och ladda upp flera SIP:ar till systemet samtidigt för vidare bearbetning och inleverans. Inleveransprocessen kan initieras genom att välja SIP:arna som ska ingå i processen. Klicka sedan på knappen "Process" för att initiera inleveransprocessen.
