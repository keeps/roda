Beziehung zu einer anderen Darstellung information

Definieren Sie eine Beziehung zwischen dieser Darstellungsinformation und einem anderen Datensatz mit Darstellungsinformationen. Der Grund für die Beziehung kann im Vokabular für den Beziehungstyp beschrieben werden, Sie können die spezifischen Darstellungsinformationen auswählen, die von diesem Repository verwaltet werden, und einen Titel für die Beziehung definieren, der als der Text des erstellten Links angezeigt wird.
