Nesta página pode consultar os autos de eliminação que foram criados para este repositório. Um auto de eliminação é composto por um relatório que agrega a informação das entidades intelectuais com informação extra(metadados). Para eliminar uma entidade intelectual associada a um auto de eliminação tem de executar explicitamente a acção de eliminar. Após eliminar as entidades intelectuais, através do auto de eliminação essas entidades podem ser restauradas ou eliminadas permanentemente.
