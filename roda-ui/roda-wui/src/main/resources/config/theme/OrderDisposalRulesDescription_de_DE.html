Auf dieser Seite ist es möglich, die Reihenfolge der Entsorgungsregeln zu ändern. Hier wird festgelegt, in welcher Priorität die Entsorgungsregeln auf das Repository angewendet werden. Die erste Position in der Tabelle entspricht der höchsten Prioritätsebene, während die letzte Position in der Tabelle der niedrigsten Prioritätsebene entspricht. Wenn der Eintrag keiner Entsorgungsregel entspricht, ist kein Entsorgungsplan zugeordnet.
