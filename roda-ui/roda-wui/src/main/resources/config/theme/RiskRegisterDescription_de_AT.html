Das Risikoregister enthält alle identifizierten Risiken, die das Repository beeinflussen können. Es sollte so umfassend wie möglich sein, um alle erkennbaren Bedrohungen zu erfassen, und im Allgemeinen eine geschätzte Wahrscheinlichkeit für das Eintreten jedes Risikos, die Schwere oder mögliche Auswirkungen des Risikos sowie seine wahrscheinliche Zeit oder erwartete Häufigkeit enthalten. Risikominderung ist der Prozess der Definition von Maßnahmen zur Verbesserung von Chancen und zur Verringerung von Bedrohungen für die Zielsetzungen des Repositories.
