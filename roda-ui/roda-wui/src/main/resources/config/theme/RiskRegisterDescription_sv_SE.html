Riskregistret listar alla identifierade risker som kan påverka systemet. Det bör vara så omfattande som möjligt och inkludera alla identifierade risker och en uppskattad sannolikhet att respektive risk inträffar, allvarlighetsgraden eller möjlig påverkan, när den kan inträffa och en uppskattning av frekvens. Riskminimeringshanteringen definierar de åtgärder som krävs för att öka möjligheten att reducera hot mot arkivobjekten.
