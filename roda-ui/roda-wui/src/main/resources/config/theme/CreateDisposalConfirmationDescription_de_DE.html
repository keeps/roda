Auf dieser Seite können Sie die intellektuellen Entitäten sehen, deren Vernichtung oder Überprüfung überfällig ist. Wenn die Unterlagen zur Vernichtung vorliegen, können sie in eine neue Entsorgungsbestätigung integriert werden oder ihr Entsorgungsplan kann angepasst werden. Wenn die Unterlagen zur Überprüfung bereit sind, können Sie nur den Entsorgungsplan ändern. Es ist auch möglich, die Unterlagen einzusehen, deren Aufbewahrungsfrist nicht berechnet wurde.
